3. MOT 6단계 (매장 고객 여정)

1

🚪 입장

행동: 아이컨택 + 밝은 인사 (3초 이내)

"어서오세요! 어떤 도움 드릴까요?"

2

⏳ 대기

행동: 예상 대기 시간 안내, 5분 이상 시 재안내 필수

"잠시만 기다려 주시면 순서대로 안내드릴게요."

3

💬 상담 시작

행동: 60초 초기 진단 질문 — 현재 통신사, 번호 유지 여부, 데이터 사용량, 중요 기준

"현재 통신사는 어디세요? 번호는 그대로 쓰실 건가요?"

4

📱 상담 진행

행동: 모델 2~3개 비교, 요금제 2개 제안, 총 비용(월 납부액 + 초기 비용) 한 번에 제시

주의: 장점만 말하지 말고 단점도 솔직하게 안내 (신뢰도 상승)

5

✍️ 마무리

행동: 개인정보 동의 + 서류 체크 + 개통 완료 확인, 사은품 지급 조건 재안내

"개통 완료 문자 오면 바로 확인해 보세요."

6

🙏 배웅

행동: 일어서서 문 쪽 배웅 (가능하면). 마지막 인상이 재방문을 결정

"감사합니다! 불편한 점 있으시면 언제든 연락 주세요."

4. 쿠션멘트 (거절·제한 상황 대응)

🛠️ 개통 실무 & 서류
신입이 가장 많이 실수하는 부분 — 한 건 한 건 체크리스트를 사용하세요.

1. 개통 전 필수 확인 10가지

  • 고객 본인 확인 (신분증 실물 확인)
  • 명의자/미성년/대리인 여부 확인 (대리 개통 규정)
  • 미납/연체/한도/신용 관련 알림 확인
  • 요금제/약정 기간 고객이 이해했는지 확인
  • 단말 색상/용량/모델 정확히 확인
  • 유심 타입 (USIM/eSIM) 확인
  • 번호이동이면 이전 통신사 해지/중복 요금 안내
  • 부가서비스는 고객 요청/필수 조건만 가입 (조건 문구 정확히)
  • 사은품 지급 조건 (요금제·부가 유지기간) 안내
  • 총 비용 (초기 + 월) 재확인

2. 서류 누락 방지 체크 (개통 건마다)

  • 신청서 서명 (필수 항목 전체)
  • 개인정보 수집/이용 동의서
  • 신분증 사본/확인 방식 (규정 준수)
  • 할부/약정 관련 고지 확인
  • 사은품 지급 내역 (수령 서명)
✅
원칙: 서류가 1장이라도 빠지면 그날 마감 전에 반드시 회수 (익일로 넘기지 않기)
