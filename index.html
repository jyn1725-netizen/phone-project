3. MOT 6단계 (매장 고객 여정)

1

🚪 입장

행동: 아이컨택 + 밝은 인사 (3초 이내)

"어서오세요! 어떤 도움 드릴까요?"

2

⏳ 대기

행동: 예상 대기 시간 안내, 5분 이상 시 재안내 필수

"잠시만 기다려 주시면 순서대로 안내드릴게요."

3

💬 상담 시작

행동: 60초 초기 진단 질문 — 현재 통신사, 번호 유지 여부, 데이터 사용량, 중요 기준

"현재 통신사는 어디세요? 번호는 그대로 쓰실 건가요?"

4

📱 상담 진행

행동: 모델 2~3개 비교, 요금제 2개 제안, 총 비용(월 납부액 + 초기 비용) 한 번에 제시

주의: 장점만 말하지 말고 단점도 솔직하게 안내 (신뢰도 상승)

5

✍️ 마무리

행동: 개인정보 동의 + 서류 체크 + 개통 완료 확인, 사은품 지급 조건 재안내

"개통 완료 문자 오면 바로 확인해 보세요."

6

🙏 배웅

행동: 일어서서 문 쪽 배웅 (가능하면). 마지막 인상이 재방문을 결정

"감사합니다! 불편한 점 있으시면 언제든 연락 주세요."

4. 쿠션멘트 (거절·제한 상황 대응)

상황❌ 안 좋은 표현✅ 쿠션멘트
요청 불가"그건 안 됩니다""죄송합니다만, 현재 정책상 어렵고, 이런 방법은 어떠세요?"
재고 없음"없어요""현재 입고 대기 중이고, ○일 이후 가능합니다. 예약해 드릴까요?"
가격 협상"가격은 못 낮춰요""저도 더 드리고 싶은데, 지원 가능한 부분을 최대한 찾아볼게요."
클레임"고객님이 잘못한 거예요""불편하셨겠어요. 내용을 확인해 보고 바로 안내드릴게요."
오래 걸림"좀 걸려요""현재 처리 중이라 약 ○분 정도 소요됩니다. 잠시만 기다려 주시겠어요?"

5. 전화응대 SOP

📞
전화응대 순서: ① 3번 울리기 전 수신 → ② "안녕하세요, 폰파는녀석들 ○○점입니다!" → ③ 용건 파악 → ④ 담당자 연결 또는 직접 해결 → ⑤ "감사합니다. 더 궁금한 점 있으시면 언제든 연락 주세요!"

주의: 고객 정보(이름·번호·주민번호)는 공개된 공간에서 큰 소리로 말하지 않기.

3. 데이터 이동 완전 가이드

⏱️
고객에게 미리 안내: "데이터 이동은 보통 30~90분 걸릴 수 있어요. 중요한 인증 앱은 재로그인이 필요합니다."

① iOS → iOS (아이폰 → 아이폰)

방법 1: iCloud 백업 복원 (권장)

  1. 이전 기기: 설정 > [이름] > iCloud > iCloud 백업 ON 확인
  2. 용량 확인 — 무료 5GB 부족 시 구매 필요 (50GB: 월 1,100원)
  3. Wi-Fi 연결 후 "지금 백업" 실행 → 완료 확인
  4. 새 기기 설정: "iCloud 백업에서 복원" 선택
  5. Apple ID 로그인 → 백업 날짜 선택 → 완료까지 대기 (10분~2시간)

방법 2: 퀵스타트 (직접 이동)

  1. 두 기기를 가까이 놓으면 "새 iPhone 설정" 팝업 자동 표시
  2. 이전 기기에서 카메라로 원형 패턴 인식
  3. 암호 입력 → Face ID/Touch ID 설정 → 데이터 이동 시작
⚠️
iCloud 백업 전 카카오톡·라인 등 메신저 앱의 별도 대화 백업을 먼저 진행하세요. iCloud만으로는 메신저 내부 데이터가 완전히 이동되지 않을 수 있습니다.

② Android → Android (삼성 → 삼성)

Samsung Smart Switch (권장)

  1. 양쪽 기기에 Smart Switch 앱 설치
  2. 연결: USB 케이블(빠름·안정적) 또는 Wi-Fi Direct(케이블 없이)
  3. 이전 기기에서 "보내기", 새 기기에서 "받기" 선택
  4. 이동할 항목 선택 (연락처·사진·앱·설정 등)
  5. 전송 시작 → 완료까지 대기 (5~60분, 용량에 따라 상이)
💡
팁: Smart Switch는 앱 데이터도 이동하지만, 보안 앱(뱅킹·카드)은 재인증이 필요합니다. 이동 후 금융앱 실행 테스트를 꼭 도와주세요.

③ Android → iOS (갤럭시 → 아이폰)

  1. 새 아이폰 설정 중 "Android에서 데이터 이동" 화면에서 시작
  2. 이전 기기(Android)에 "iOS로 이전" 앱 설치
  3. 새 아이폰에서 표시되는 12자리 코드를 이전 기기에 입력
  4. Wi-Fi 자동 연결 → 이동할 항목 선택 → 완료까지 대기

이동 불가 데이터: 카카오톡 대화 내용(별도 백업 필수), 보안 앱(재인증 필요)

④ iOS → Android (아이폰 → 갤럭시)

  1. 새 기기에 Samsung Smart Switch 설치
  2. 케이블 또는 Wi-Fi로 연결 → 아이폰에서 "보내기" 선택
  3. 또는 Smart Switch의 "iCloud에서 가져오기"로 백업 데이터 이동 가능

⑤ 카카오톡 대화 백업 & 복원

⚠️
카카오톡 대화는 기기에만 저장됩니다. 이동 전 반드시 별도 백업이 필요합니다.
플랫폼백업 경로저장 위치

복원: 새 기기 카카오톡 설치 → 기존 계정 로그인 → "대화 내용 복원" → 비밀번호 입력 → 복원 완료 (10~60분 소요)
